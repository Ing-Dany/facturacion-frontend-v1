Brief del Ejercicio — Sistema de Facturación

Taller Práctico | Desarrollo con IA usando Metodología SDD

1. Descripción del Ejercicio

Construir un sistema web de facturación para una empresa pequeña en Ecuador, aplicando la metodología Spec-Driven Development (SDD) con asistencia de IA. El sistema se desarrollará de punta a punta: desde la especificación hasta el deploy en producción.

2. Módulos del Sistema

Módulo Funcionalidad Detalle
Usuarios Registro, Login, Logout 1 solo rol, autenticación JWT
Clientes CRUD completo Validación cédula/RUC Ecuador
Productos CRUD completo + Stock Precio unitario, flag IVA, cantidad en stock
Facturas Crear factura con detalle Cabecera + líneas de detalle, cálculo automático
Reportes Consultas y listados Ventas por período, por cliente, productos más vendidos

3. Reglas de Negocio Principales

  • IVA: 15% (tarifa vigente Ecuador). Se aplica solo a productos marcados con IVA.
  • Cálculo: Subtotal 0% + Subtotal 15% + IVA = Total
  • Numeración: Secuencial automática (FAC-000001, FAC-000002...)
  • Estados: EMITIDA → ANULADA (no se eliminan facturas)
  • Autenticación: JWT con expiración de 8 horas

4. Alcance por Versión

v1 — Lo que construimos en el taller:
  • CRUD completo de Usuarios, Clientes y Productos
  • Creación y anulación de facturas con detalle
  • Reportes básicos
  • Al crear factura NO se resta stock del producto
v2 — Ejercicio final (mejora):
  • Al crear factura se descuenta automáticamente el stock del producto
  • Validación de stock suficiente antes de emitir

6. Metodología: Spec-Driven Development (SDD)

La especificación es la fuente de verdad. El flujo de trabajo es:

  1. Especificar — Definir qué se construye (historias, flujos, reglas)
  2. Diseñar — Arquitectura y modelo de datos (C4, DDL)
  3. Implementar — La IA genera código a partir de la spec
  4. Validar — Tests y verificación contra la spec
  5. Desplegar — Push y deploy automático

7. Lo que NO incluye el ejercicio

  • Sistema de inventario completo (solo campo stock)
  • Múltiples roles o permisos
  • Recuperación de contraseña (requiere servicio de email)
  • Generación de PDF
  • Facturación electrónica (SRI)
